Android 개발 환경 설정하기

1. Android Studio란?

Android Studio는 구글에서 공식으로 지원하는 Android용 앱 개발을 위한 통합 개발 환경 (IDE, Integrated Development Environment)입니다.


1) Android Stuoid 설치

파일 다운로드 아래 링크로 이동하여 안드로이드 스튜디오를 다운로드한다.

✔️ 안드로이드 스튜디오 다운로드

  • 링크로 이동하여 아래 캡처화면처럼 빨간 네모로 표시된 Download Android Studio를 클릭합니다.
  • 약관을 읽고 동의 한 뒤 Download Android Stuio *** for Windows 버튼을 클릭해 안드로이드 스튜디오 설치 파일을 다운로드합니다.

2) Android Studio 설치

1단계에서 다운로드 받은 설치 파일을 실행해 안드로이드 스튜디오를 설치합니다.

  • Next 버튼을 클릭하여 설치를 진행합니다.
  • ❗ 아래 화면에서 Android Virtual Device를 체크해야 안드로이드가 제공하는 에뮬레이터를 사용할 수 있습니다. 옵션 체크 후 Next 버튼을 클릭합니다.
  • 안드로이드 스튜디오가 설치될 경로를 설정합니다. 이번 환경 설정에서는 기본 경로 그대로 진행하였습니다. Next 버튼을 클릭합니다.
  • 아래 캡처 화면은 시작메뉴에 표시될 폴더 이름을 설정하는 단계입니다. Android Studio 그대로 설정하고 Install 버튼을 클릭해 설치를 진행했습니다.
  • Install이 완료된 후 Finish 버튼을 클릭하여 설치를 완료합니다.

3) Android Studio 설정

앞의 과정에서 설치한 안드로이드 스튜디오를 실행하여 기본 설정을 진행합니다.

  • 기본 설정 정보를 읽어 드릴지 선택하는 화면입니다. Do not import settings를 선택하여 초기 상태로 진행하였습니다.
  • Next 버튼을 클릭하여 안드로이드 스튜디오 개발 환경 설정을 수행합니다.
  • 개발환경 설정 방식을 선택합니다. Standard는 안드로이드 스튜디오에서 추천하는 표준 방식이고, Custom은 사용자가 직접 설정할 수 있습니다. Standard를 선택한 뒤 Next 버튼을 클릭해 표준 방식으로 진행하였습니다.
  • UI 테마를 선택할 수 있습니다. Darcula 테마를 선택했습니다.
  • 설정한 개발 환경을 최종적으로 확인한 뒤 Finish 버튼을 클릭해 환경 설정을 완료합니다.

이제 설치한 안드로이드 스튜디오를 실행하여 프로젝트를 생성하고 안드로이드 네이티브 앱 개발을 수행할 수 있습니다.
